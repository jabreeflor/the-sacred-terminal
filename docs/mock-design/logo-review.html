Marks for review — including a Sacred Timeline direction from your reference. Preview canvas is white so dark app squares read clearly. File: docs/mock-design/logo-review.html

Your reference. Glowing fractal branches from a horizontal trunk, electric blue with magenta filaments, viewed through a dark circular portal. App icons below simplify that into vector marks that still read at 16 px.

Brief. Sacred tree icons use cyan #5dd4ff, blue #2a8cff, magenta #e879c9 on deep navy #060612. Orange terminal accent stays separate in the app UI.

Recommendation

For the reference look: N · Timeline as the primary Dock icon (tree reads clearly at 32 px). Use M · Portal for marketing / about screen where the circular viewport can breathe.

  • N — best balance for macOS app icon + titlebar
  • O — fallback if N feels busy in the Dock
